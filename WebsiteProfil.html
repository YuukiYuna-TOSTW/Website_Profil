Skill

HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun struktur halaman web. HTML bekerja dengan menggunakan tag-tag untuk menentukan elemen-elemen yang tampil di browser. HTML tidak memiliki kemampuan logika seperti bahasa pemrograman, tetapi berfungsi sebagai fondasi utama dalam membangun konten dan tata letak halaman web, yang kemudian dapat dikombinasikan dengan CSS untuk tampilan dan JavaScript untuk interaktivitas.`

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan gaya dari elemen-elemen HTML di halaman web. Dengan CSS, kita dapat mengatur warna, ukuran, jenis huruf, jarak, posisi, tata letak, dan berbagai aspek visual lainnya dari suatu elemen. CSS memungkinkan pemisahan antara struktur (HTML) dan presentasi, sehingga desain web menjadi lebih fleksibel dan mudah dikelola. CSS dapat ditulis langsung di dalam tag HTML (inline) yang kemudian ke file utama berisikan HTML.

Java Script

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan dinamika pada halaman web. Dengan JavaScript, elemen HTML dapat dimanipulasi secara langsung, seperti menangani klik tombol, mengubah konten secara real-time, membuat animasi, memvalidasi formulir, dan berkomunikasi dengan server tanpa memuat ulang halaman (AJAX). JavaScript dijalankan di sisi klien (browser) dan bekerja bersama HTML dan CSS untuk menciptakan pengalaman pengguna yang responsif dan interaktif. Selain di browser, JavaScript juga dapat digunakan di server melalui platform seperti Node.js.

Python

Python adalah bahasa pemrograman tingkat tinggi yang dirancang dengan sintaks yang sederhana dan mudah dibaca, sehingga cocok untuk pemula maupun profesional. Python bersifat serbaguna dan dapat digunakan untuk berbagai bidang seperti pengembangan web, analisis data, kecerdasan buatan, otomatisasi, hingga pengembangan perangkat lunak. Bahasa ini mendukung berbagai paradigma pemrograman, termasuk pemrograman berorientasi objek dan fungsional. Selain itu, Python memiliki komunitas yang besar dan pustaka (library) yang sangat kaya, seperti NumPy untuk sains data, Django untuk web, dan TensorFlow untuk machine learning.

Java

Java adalah bahasa pemrograman berorientasi objek yang dirancang untuk dapat dijalankan di berbagai platform melalui prinsip “tulis sekali, jalankan di mana saja” (write once, run anywhere). Program Java dikompilasi menjadi bytecode yang dijalankan oleh Java Virtual Machine (JVM), sehingga dapat berjalan di berbagai sistem operasi tanpa perlu diubah. Java banyak digunakan dalam pengembangan aplikasi desktop, web, mobile (terutama Android), dan sistem enterprise. Bahasa ini dikenal stabil, aman, dan memiliki struktur yang kuat, serta didukung oleh banyak pustaka dan framework seperti Spring dan JavaFX.

Tailwind CSS

Tailwind CSS adalah framework CSS utility-first yang memungkinkan pengembang membangun antarmuka pengguna secara cepat dan efisien dengan menggunakan kelas-kelas siap pakai langsung di elemen HTML. Alih-alih menulis CSS kustom untuk setiap komponen, Tailwind menyediakan utilitas seperti `p-4` untuk padding, `text-center` untuk rata tengah teks, atau `bg-blue-500` untuk latar belakang biru, sehingga styling dapat dilakukan langsung di dalam markup. Pendekatan ini membantu mengurangi penulisan CSS yang berulang, menjaga konsistensi desain, dan mempercepat proses pengembangan antarmuka. Tailwind juga mendukung kustomisasi penuh melalui file konfigurasi dan integrasi dengan berbagai tools modern.

Bootstrap

Bootstrap CSS adalah framework front-end yang dirancang untuk mempercepat dan mempermudah pengembangan antarmuka web yang responsif dan konsisten. Dikembangkan oleh tim dari Twitter, Bootstrap menyediakan kumpulan kelas dan komponen siap pakai seperti tombol, formulir, navigasi, dan grid system berbasis 12 kolom yang memudahkan pengaturan tata letak. Dengan Bootstrap, pengembang dapat dengan cepat membuat desain yang menarik tanpa menulis banyak kode CSS dari awal. Framework ini juga responsif secara default, artinya tampilan akan otomatis menyesuaikan dengan ukuran layar perangkat, seperti desktop, tablet, atau ponsel.

Tools

Figma

Figma adalah alat desain antarmuka berbasis web yang digunakan untuk membuat prototipe, wireframe, dan desain UI/UX secara kolaboratif. Berbeda dengan software desain tradisional, Figma memungkinkan banyak pengguna bekerja secara real-time dalam satu proyek, mirip seperti Google Docs untuk desain. Figma mendukung berbagai fitur seperti desain vektor, sistem grid, komponen yang dapat digunakan ulang, serta integrasi prototipe interaktif tanpa perlu coding. Karena berbasis cloud, Figma dapat diakses langsung melalui browser tanpa perlu instalasi, dan tersedia juga versi desktop untuk kemudahan penggunaan.

GitHub

GitHub adalah platform berbasis web yang digunakan untuk menyimpan, mengelola, dan melacak perubahan kode program secara kolaboratif menggunakan sistem kontrol versi Git. Dengan GitHub, pengembang dapat bekerja bersama dalam satu proyek, melakukan versi kontrol, membuat cabang (branch), mengajukan perubahan melalui pull request, dan melakukan review kode secara efisien. GitHub juga menyediakan fitur dokumentasi, pelacakan isu (issues), serta integrasi dengan berbagai tools pengembangan. Selain sebagai tempat kolaborasi, GitHub juga berfungsi sebagai portofolio online bagi pengembang untuk memamerkan proyek-proyek mereka.

VS Code

Visual Studio Code (VS Code) adalah text editor sumber terbuka buatan Microsoft yang ringan namun sangat powerful, digunakan untuk menulis dan mengedit kode dalam berbagai bahasa pemrograman seperti JavaScript, Python, Java, C++, dan lainnya. VS Code mendukung fitur-fitur canggih seperti penyorotan sintaks (syntax highlighting), IntelliSense (auto-complete pintar), debugging langsung, Git integration, serta terminal bawaan. Selain itu, VS Code sangat fleksibel karena bisa diperluas dengan berbagai ekstensi yang tersedia di marketplace, memungkinkan pengguna menyesuaikan editor sesuai kebutuhan proyek. VS Code tersedia di berbagai sistem operasi seperti Windows, macOS, dan Linux.

Draw.io

draw.io adalah aplikasi berbasis web yang digunakan untuk membuat berbagai jenis diagram secara mudah dan gratis, seperti flowchart, diagram jaringan, diagram organisasi, mind map, dan diagram teknis lainnya. Dengan antarmuka drag-and-drop yang intuitif, pengguna dapat membuat, mengedit, dan menyimpan diagram langsung di browser tanpa perlu instalasi software tambahan. draw\.io juga mendukung integrasi dengan layanan penyimpanan cloud seperti Google Drive, OneDrive, dan Dropbox, sehingga memudahkan kolaborasi dan akses dari berbagai perangkat. Aplikasi ini populer di kalangan profesional dan pelajar untuk merancang visualisasi ide dan proses secara efisien.

Flowgorithm

Flowgorithm adalah aplikasi gratis yang digunakan untuk membuat dan menjalankan diagram alir (flowchart) sebagai alat bantu belajar pemrograman. Dengan Flowgorithm, pengguna bisa merancang logika program secara visual menggunakan simbol-simbol standar flowchart seperti proses, keputusan, input/output, dan loop, tanpa perlu menulis kode secara langsung. Setelah flowchart dibuat, Flowgorithm dapat menerjemahkannya menjadi kode dalam beberapa bahasa pemrograman populer seperti Python, Java, dan C++. Alat ini sangat berguna untuk pemula agar memahami konsep dasar algoritma dan struktur program secara intuitif dan interaktif.

MySQL Workbench

MySQL Workbench adalah aplikasi visual yang digunakan untuk merancang, mengelola, dan mengoptimalkan basis data MySQL secara grafis. Dengan MySQL Workbench, pengguna dapat membuat diagram Entity-Relationship (ER), menulis dan menjalankan query SQL, memonitor performa database, serta melakukan administrasi server MySQL dengan mudah melalui antarmuka yang intuitif. Aplikasi ini sangat membantu pengembang dan administrator database dalam mendesain skema, mengelola data, serta mendiagnosis masalah tanpa perlu menggunakan baris perintah secara langsung. MySQL Workbench tersedia untuk berbagai platform seperti Windows, macOS, dan Linux.
